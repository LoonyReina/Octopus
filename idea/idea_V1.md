# OpenClaw 模拟宠物系统咨询提示词（强约束版）

你是一名资深 **AI Agent / 多模态系统 / 机器人认知架构 / 长期记忆系统** 设计专家。  
请你从**系统架构、工程落地、模块边界、优先级设计**角度回答我的问题，而不是只给泛泛建议。

---

## 我的项目背景

我想基于 **OpenClaw** 做一个“**模拟宠物**”系统。  
目标不是做一个普通多模态问答助手，而是做一个能长期陪伴、认主人、学动作、形成互动习惯的 agent。

### 我希望它具备这些能力

1. **理解视频 + 语音/文字**
   - 视频以**抽帧**方式做理解
   - 语音可先转文字，但我也希望利用说话人、语气等信息
   - 最终实现视频、语音、文字联合理解

2. **记住主人**
   - 记住主人的样子、脸、声纹、说话方式
   - 区分主人、家庭成员、陌生人
   - 形成长期稳定的主人档案

3. **记住主人教过的动作**
   - 例如：坐下、握手、转圈、过来
   - 不只是“记住一句话”，而是真正形成某种技能映射
   - 能处理重复教学、纠错、奖励、强化

4. **像宠物一样长期互动**
   - 有短期状态和长期关系变化
   - 能根据情境、情绪、精力、亲密度产生不同反应
   - 不是只会等我发问再答
